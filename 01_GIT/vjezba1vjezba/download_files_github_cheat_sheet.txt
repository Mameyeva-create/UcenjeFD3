ШПАРГАЛКА ПО GIT & GITHUB DESKTOP
=================================

1. PUSH (Пуш / Отправить)
   - Толкает ваши локальные коммиты в удаленный репозиторий на GitHub.
   - Ассоциация: Выталкивать коробку со своего компьютера в облако.

2. PULL (Пул / Получить)
   - Тянет изменения из удаленного репозитория GitHub на ваш компьютер.
   - Ассоциация: Притягивать чужие обновления к себе.

3. ВЛОЖЕННЫЕ РЕПОЗИТОРИИ
   - Никогда не создавайте репозиторий внутри другого репозитория.
   - Если создали случайно, удалите скрытую папку .git во вложенной папке,
     чтобы она стала обычной частью главного проекта.
