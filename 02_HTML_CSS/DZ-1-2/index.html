POVIJEST HTML-A

OD JEDNOSTAVNOG TEKSTA DO MODERNOG WEBA

Sve je počelo 1989. godine u CERN-u. Tim Berners-Lee je osmislio sustav koji bi istrsživačima omogućio dijeljenje dokumenata. Taj sustav postao je poznat kao World Wide Web.

Otac weba: Sir Tim Berners-Lee

Jezik koji je definirao strukturu tih dokumenata nazvan je HTML (HyperText Markup Language). Njegova osnovna svrha bila je semantički opisati strukturu dokumenta, a ne njegov izgled.

Ključne verzije

  1. 01.HTML 1.0 (1991)

    Prva neslužbena verzija. Sadržavala je samo 18 tagova.

  2. 02.HTML 2.0 (1995)

    Prvi službeni standard (RFC 1866).

  3. 03.HTML 3.2 (1997)

    Uvedene tablice i apleti. "Rat preglednika" je bio u punom jeku.

  4. 04.HTML 4.01 (1999)

    Dominantna verzija gotovo desetljece. Odvajanje stila (CSS) od sadržaja.

  5. 05.HTML5 (2014)

    Revolucija. Multimedija bez dodataka ('video', 'audio'), semantički elementi.

Struktura Dokumenta

Osnovna struktura HTML5 dokumenta izgleda ovako:

<!DOCTYPE html>
        <html lang="hr">
        <head>
        <title>Naslov</title>
        </head>
        <body>
        <p>Pozdrav Svijete!</p>
        </body>
        </html>

Da biste spremili ovaj kod, pritisnite Ctrl + S na tipkovnici.

Važni pojmovi

HTTP HyperText Transfer Protocol - protokol za prijenos podataka.

URI Uniform Resource Identifier - adresa resursa na webu.
